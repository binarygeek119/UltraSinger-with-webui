Lyrics compare

Enter artist and song title to look up lyrics online (LRCLIB, lyrics.ovh, and Shazam track pages), and optionally pick an exported song to compare with your UltraStar / YARG notes.txt. The UltraSinger column merges multi-note syllables into one line per melisma run (using ~ in the file to decide where each run ends); each note is separated by a space in that view, and a trailing ~ becomes an extra space. The file and the correction box stay one line per note and keep ~ as stored. Reference does the same for trailing tildes in the view only. Reference chips: chips are always one word per button (newlines are treated as whitespace). Chips are aligned to the matching UltraSinger note/run (skipping letter-spellings like a+n+d so And maps to the real word later in the file). Saving writes corrections back to the note file.

Reference (databases)

Fetch results appear after “Load comparison”. Chips are tinted green when they match the merged UltraSinger syllable, red when they differ (case-insensitive; trailing ~ ignored). Click a word to set that syllable. Custom paste below updates the clickable words.
